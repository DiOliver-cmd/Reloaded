# Reloaded
## 📄 Sobre o Projeto

O **Piscine Reloaded** é um projeto fundamental da 42 São Paulo criado para relembrar, consolidar e testar os conceitos básicos aprendidos durante a Piscine. Ele serve como uma ponte de aquecimento antes de iniciar os desafios do *Common Core*.

O repositório contém uma série de exercícios práticos que revisitam comandos de terminal do ecossistema Unix, criação de scripts em Shell e a transição lógica para a linguagem C (incluindo loops, funções básicas e o uso de ponteiros).

---

## 🛠️ Conteúdo do Repositório

Os exercícios são divididos logicamente entre manipulação do sistema e lógica de programação:

### 🐚 Comandos de Terminal & Shell Script
*   **Exercícios iniciais:** Manipulação de arquivos, permissões de sistema (`chmod`), criação de links simbólicos e buscas avançadas com `find`.
*   **Automação:** Criação de scripts básicos para automatizar tarefas simples no terminal.

### 💻 Fundamentos da Linguagem C
*   **Estruturas básicas:** Exibição de caracteres, manipulação de loops e condições.
*   **Lógica de ponteiros:** Implementação de funções clássicas como `ft_ft` e `ft_ultimate_ft` para compreender o mapeamento e alteração de valores na memória.
*   **Manipulação de strings e arrays:** Recriação de funções utilitárias como `ft_strlen`, `ft_strcmp` e manipulação de arrays com `ft_foreach`.

---

## 📐 Padrão de Codificação (Norminette)

Todos os arquivos em C deste projeto foram desenvolvidos seguindo rigidamente a **Norminette**, o guia de estilo e boas práticas da 42.
*   Limite de 25 linhas por função.
*   Declarações de variáveis alinhadas.
*   Proibição de estruturas de controle de fluxo de forma aninhada excessiva ou fora do padrão.
